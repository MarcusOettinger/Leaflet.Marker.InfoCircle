Leaflet.Marker.InfoCircle

A simple leaflet marker with a tiny bit of text in the center.

View on GitHub Download .zip Download .tar.gz

Leaflet.Marker.InfoCircle is a dead simple leaflet plugin for a marker similar to L.CircleMarker to display a string or number in the circle.

Usage

Just include js and css files

<script src="Leaflet.Marker.InfoCircle.js"></script>
<link href="Leaflet.Marker.InfoCircle.css" rel="stylesheet"/>

and use the new marker with some new options (this is the first entry in the example above): 

  var info1 = new L.Marker.InfoCircle( map.getCenter(), {
          text: "foo", 
          circleClass: "circle1"
  }).addTo(map);
  info1.bindPopup("<h2>A popup</h2>(you guessed that...)<br>").openPopup();

Options

Leaflet.Marker.InfoCircle brings some new options to be set in the constructor:

  • text: the string displayed in the marker
  • radius: radius of the circle in px
  • borderWidth: width of the border in px
  • circleClass: a class to set to the marker to allow for css styling
