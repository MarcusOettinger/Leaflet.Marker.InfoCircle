Leaflet.Marker.InfoCircle

A simple leaflet marker with a tiny bit of text in the center.

View on GitHub Download .zip Download .tar.gz

Leaflet.Marker.InfoCircle is a dead simple leaflet plugin for a marker similar to L.CircleMarker to display a string or number in the circle.

Usage

Just include js and css files

Hallo

and use the new marker with some new options (this is the first entry in the example below): 

  var info1 = new L.Marker.InfoCircle( map.getCenter(), {text: "foo", circleClass: "circle1"}).addTo(map)
